ZeroSprites is a CSS sprites generator aimed at area minimization using algorithms used in the field of VLSI floorplanning.
It's a bash script using two VLSI floorplanning tools and ImageMagick.

Github: https://github.com/clyfish/zerosprites

Email: <my github username> at gmail.com

  • Tools:
  • rectpack (Huang and Korf) finds optimal solutions when there are not so many images.
  • btree (T.-C. Chen and Y.-W. Chang, no source code) Find sub-optimal solutions when there are too many images.
  • ImageMagick generates sprites sheets according to the output of the above two tools.
  • pngnq generates png8 images.
  • pngopt optimizes the size of png files.
  • Bootstrap styles this website.
  • Papers:
  • T.-C. Chen and Y.-W. Chang, ``Modern Floorplanning Based on B*-trees and Fast Simulated Annealing", IEEE Trans. Computer-Aided Design of Integrated Circuits and Systems (TCAD), Vol. 25, No. 4, pp. 637--650, April 2006.
  • Korf R. and Huang E., ``Optimal rectangle packing: an absolute placement approach", Journal of Artificial Intelligence Research, Vol. 46, P. 47-87, 2012.
